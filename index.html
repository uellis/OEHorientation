Database searching

To find articles on a topic, youâ€™ll need to search a database like Medline. It helps to break your question into main concepts before you search. One way to do this is with PICO:

  • Patient/Population/Problem
  • Intervention/Issue
  • Comparison (if applicable)
  • Outcomes

PICO example

In patients with brain injury, does workload affect successful return to work?

P - brain injury, brain injuries, TBI, ABI, MTBI, concussion

I - workload, work load, duties, volume of work

C - (none stated)

O - return to work, RTW, retention, back to work, remain at work

Think about

  • singular/plural/different word endings
  • North American/UK spellings
  • Acronyms
  • Antonyms (eg, falls vs balance; mental disorder vs mental health)
